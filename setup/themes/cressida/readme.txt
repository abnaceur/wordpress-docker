== Description ==

Cressida is a feature-packed, gorgeous blogging theme perfect for a variety of uses including food, personal, and lifestyle blogs. 

* Front Page Banner
* Front Page Posts Slider 
* Front Page Featured Posts
* Front Page Highlight Post
* Front Page Promo Category
* Front Page Posts Strip
* Front Page Featured Pages
* Easy Social Media Icons / Menus
* Multiple Sidebars
* Multiple Footer Sidebars
