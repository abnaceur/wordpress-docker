== Description ==

GTL News Theme is a simple, beautifully designed responsive WordPress News & Magazine theme. GTL News theme is especially for  News and Blog theme. Fully responsive, cross-browser compatible, SEO friendly, GTL News Theme is news and magazine theme and is suitable for news site, blog site , magazine site . We support via online chat, email, support forum.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Adding Widgets In Front Page ==

1. Add widget 'GTL Main Highlight' in 'Home Page Main Highlight' widget area.
2. Add widget 'GTL Feature' in 'Home Page Secondary' widget area.
3. Add widget 'GTL Feature  Two' in 'Home Page Secondary' widget area.
4. Add widget 'GTL Recent' in 'Home Page Sidebar' widget area.
